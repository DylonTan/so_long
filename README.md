# so_long
A 2D game made with minilibx, a bare-bones graphics library

## Getting started
How to compile:
### `make all`

How to compile bonus part:
### `make bonus`

How to recompile:
### `make re`

How to clean:
### `make clean`

How to do a full clean:
### `make fclean`
